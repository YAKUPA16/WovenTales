# WovenTales

**WovenTales** is an interactive storytelling web application where users collaboratively create, explore, and experience branching narratives. Users can contribute to stories, make choices that influence the plot, and enjoy a dynamic storytelling experience.

## Features

- **Collaborative Storytelling:** Multiple users can contribute to a single story.
- **Branching Narratives:** Each decision can lead to different story paths and endings.
- **User-Friendly Interface:** Simple and engaging UI for creating and exploring stories.
- **Save Progress:** Users can save and resume their stories.
- **Responsive Design:** Works well on both desktop and mobile devices.

## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
- **Version Control:** Git + GitHub
